#### UNIFAC
- Some history - Broad overview over the model - Fomulas - Required data - Result accuracy -> paper examples - Limitation: Assumes VLE!

The UNIFAC model

  • Calc pure component parameters based on functional groups
  • Calc component parameters in the mixture based on pure components
  • Calc activity coefficients:
    • Combinatorial part: From component
    • Residual part: From component and interaction

The maths

Volume and surface area parameters per component: \begin{aligned} r_i &= \sum_j R_j \nu_j^{(i)} \\ q_i &= \sum_j Q_j \nu_j^{(i)} \end{aligned}
Volume and surface area parameters in the mixture per component: \begin{aligned} \phi_i &= \frac{ r_i }{ \sum_j r_j x_j } \\ \theta_i &= \frac{ q_i }{ \sum_j q_j x_j } \end{aligned}
Combinatorial part: \begin{aligned} \ln \gamma_i^C = 1 - \phi_i' + \ln \phi_i' - \frac{z}{2} q_i \left( 1 - \frac{\phi_i}{\theta_i} + \ln \frac{\phi_i}{\theta_i} \right) \end{aligned}
Group surface area fraction per component: \begin{aligned} \Theta^{(i)}_k = \frac{ Q_k X_k^{(i)} }{ \sum_n Q_n X_n^{(i)} } \end{aligned}
Group activity coefficients per component:
\begin{aligned} \ln \Gamma_k^{(i)} = Q_k \cdot \left( 1 - \ln \left( \sum_m \theta_m^{(i)}\Psi_{mk} \right) - \sum_m \frac{ \theta_m^{(i)} \Psi_{km} }{ \sum_n \theta_n^{(i)} \Psi_{nm} } \right) \end{aligned} Nochmal gucken, letzter Buch Indices Zähler/Nenner passt nicht Group interaction parameters: \begin{aligned} \Psi_{ij} = \exp \left( - \frac{ a_{ij} + b_{ij} T + c_{ij} T^2 }{ T } \right) \end{aligned} Nochmal gucken, ob das nicht UNIFAC-DMD ist (Anz. Parameter)
Group mole fractions in the mixture \begin{aligned} X_i = \frac{ \sum_j x_j \nu_i^{(j)} }{ \sum_j x_j \sum_k \nu_k^{(j)} } \end{aligned} Group surface area fraction: \begin{aligned} \Theta_k = \frac{ Q_k X_k }{ \sum_n Q_n X_n } \end{aligned}
Group activity coefficients \begin{aligned} \ln \Gamma_k = Q_k \cdot \left( 1 - \ln \left( \sum_m \theta_m \Psi_{mk} \right) - \sum_m \frac{ \theta_m \Psi_{km} }{ \sum_n \theta_n \Psi_{nm} } \right) \end{aligned} Nochmal gucken, letzter Buch Indices Zähler/Nenner passt nicht
Residual part: \begin{aligned} \ln \gamma_i^R = \sum_k \nu_k^{(i)} \left( \ln \Gamma_k - \ln \Gamma_k^{(i)} \right) \end{aligned} Activity coefficient: \begin{aligned} \ln \gamma_i = \ln \gamma_i^C + \ln \gamma_i^R \end{aligned}
