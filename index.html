#### UNIFAC

The UNIFAC model

  • Calc pure component parameters based on functional groups
  • Calc component parameters in the mixture based on pure components
  • Calc activity coefficients:
    • Combinatorial part: From component
    • Residual part: From component and interaction

The maths

Volume and surface area parameters per component: \begin{aligned} r_i &= \sum_j R_j \nu_j^{(i)} \\ q_i &= \sum_j Q_j \nu_j^{(i)} \end{aligned}
Volume and surface area parameters in the mixture per component: \begin{aligned} \phi_i &= \frac{ r_i }{ \sum_j r_j x_j } \\ \theta_i &= \frac{ q_i }{ \sum_j q_j x_j } \\ l_i &= \frac{z}{2} (r_i-q_i) - (r_i - 1) \end{aligned}
Combinatorial part: \begin{aligned} \ln \gamma_i^C = \ln \frac{\phi_i}{x_i} + \frac{z}{2} q_i \ln \frac{\theta_i}{\phi_i} + l_i - \frac{\phi_i}{x_i} \sum_j x_jl_j \end{aligned}
Group mole fractions in pure components \begin{aligned} X_m^{(i)} = \frac{\nu_m^{(i)}}{\sum_n \nu_n^{(i)}} \end{aligned} Group mole fractions in the mixture \begin{aligned} X_m = \frac{ \sum_j x_j \nu_m^{(j)} }{ \sum_j \sum_m x_j \nu_m^{(j)} } \end{aligned}
Group surface area fraction: \begin{aligned} \Theta_k = \frac{ Q_k X_k }{ \sum_n Q_n X_n } \end{aligned} Group surface area fraction per component: \begin{aligned} \Theta^{(i)}_k = \frac{ Q_k X_k^{(i)} }{ \sum_n Q_n X_n^{(i)} } \end{aligned}
Group interaction parameters: \begin{aligned} \Psi_{ij} = \exp \left( - \frac{a_{ij}}{T} \right) \neq \Psi_{ji} \end{aligned}
Group activity coefficients per component: \begin{aligned} \ln \Gamma_k^{(i)} = Q_k \cdot \left( 1 - \ln \left( \sum_m \theta_m^{(i)}\Psi_{mk} \right) - \sum_m \frac{ \theta_m^{(i)} \Psi_{km} }{ \sum_n \theta_n^{(i)} \Psi_{nm} } \right) \end{aligned} Group activity coefficients in the mixture: \begin{aligned} \ln \Gamma_k = Q_k \cdot \left( 1 - \ln \left( \sum_m \theta_m\Psi_{mk} \right) - \sum_m \frac{ \theta_m \Psi_{km} }{ \sum_n \theta_n \Psi_{nm} } \right) \end{aligned}
Residual part: \begin{aligned} \ln \gamma_i^R = \sum_k \nu_k^{(i)} \left( \ln \Gamma_k - \ln \Gamma_k^{(i)} \right) \end{aligned} Activity coefficient: \begin{aligned} \ln \gamma_i = \ln \gamma_i^C + \ln \gamma_i^R \end{aligned}

Limitations

  • Since the model relies on empirical data, it is not 100% accurate
  • The given mixture having a VLE state is a requirement, the model does not validate the VLE
  • The model only predicts for substances which can be describes as a combination of fuctional groups with available data
